# React.js sample

## Description

じゃんけん (Rock-paper-scissors) game

* Ruby on Rails backend
* React.js frontend
* Redux container
	* Redux Thunk middleware 
* Material Design Lite CSS/JS

## Requirements

* Ruby ~> 2.3
* Bundler ~> 1.11
* Node.js ~> 6.2
* npm ~> 3.8
* webpack ~> 1.13
